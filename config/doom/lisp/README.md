# Private Lisp Library

## Overview

This directory contains private, custom Emacs Lisp packages and libraries that extend the functionality of Doom Emacs with robust, self-contained utility functions.

## Files

*   **[cashpw-time.el](file:///google/src/cloud/cashweaver/emacs/google3/experimental/users/cashweaver/dotfiles/files/doom/lisp/cashpw-time.el)**: A standalone utility package providing robust constants and helper functions for date/time calculations, ISO week conversions, and temporal predicates.
*   **[cashpw-time-test.el](file:///google/src/cloud/cashweaver/emacs/google3/experimental/users/cashweaver/dotfiles/files/doom/lisp/cashpw-time-test.el)**: An extensive ERT regression test suite that thoroughly exercises all date and time utilities in `cashpw-time.el` under deterministic, mocked temporal environments.
*   **[org-buganizer-extras.el](file:///google/src/cloud/cashweaver/emacs/google3/experimental/users/cashweaver/dotfiles/files/doom/lisp/org-buganizer-extras.el)**:
    Extras and custom extensions for org-buganizer, keeping configuration
    modular.
*   **[org-buganizer-extras-test.el](file:///google/src/cloud/cashweaver/emacs/google3/experimental/users/cashweaver/dotfiles/files/doom/lisp/org-buganizer-extras-test.el)**:
    A comprehensive unit testing suite for the custom org-buganizer-extras
    extensions.
